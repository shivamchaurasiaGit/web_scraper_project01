# Python Web Scraper Project

This project is a Python-based web scraper that extracts book names and prices from a website and saves them into a CSV file.

## Technologies Used
- Python
- Requests
- BeautifulSoup

## Features
- Scrapes product data
- Saves output to CSV
- Simple and beginner-friendly automation script

## Use Case
Useful for automation, data collection, and entry-level Python projects.
